## Roman Orthography Notes
The standardized Nattilik Roman orthography can be represented by the same standardized roman orthography for all of Nunavut Inuk dialect communities (inuktut qaliujaaqpait, "inuktut roman orthography"), however, Nattilingmiutut uses a different character representation in it's standardized roman orthography for the distinct Nattilik dialect sounds.

Full Nattilik Roman Orthography Character Set:
> i u a pi pu pa ti tu ta ki ku ka gi gu ga mi mu ma ni nu na si su sa li lu la ji ju ja vi vu va ri ru ra qi qu qa ngi ngu nga ≈Çi ≈Çu ≈Ça hi hu ha ≈ôi ≈ôu ≈ôa ≈°i ≈°u ≈°a chi chu cha

‚ù∏ Please note that there are two existing practices within the Nattilingmiutut community towards representing the roman transliteration of the angma (·ñï) character. While the standardized inuktut qaliujaaqpait uses the digraph "ng" to represent "·ñï", some Nattilik writers represent "·ñï" using `≈ã`. For this reason, this character is required in the Nattilik keyboard layout to accommodate this transliteration practice in the community.
